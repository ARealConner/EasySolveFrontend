The interface consists of two text boxes. The left textbox is for input where you'll define variables and equations, while the right textbox will display the output.

Real-time Interaction

EasySolve.AI updates in real-time as you type. It supports some latex characters and that allows you to type using \ followed by the corresponding symbol name such as, \alpha which turns into its corresponding symbol α.
